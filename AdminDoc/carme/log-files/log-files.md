# How to check the Carme log files

The log files are stored in the login-node:

1.  `ssh persephone`
2.  `cd /var/log/`

## For Hadez (acheron):
3. `cd Carme--Hadez`
4. `cd Carme-Apache-Logs`
You can access `django.log` and `error.log`

## For Hades (styx):
5. `cd Carme`
6. `cd Carme-Apache-logs`
You can access `django-log` and `error.log`
